# vue-elegant-button

## Project setup

```
npm install --save my-button
```

or

```
yarn add my-button
```

## Basic Usage

```
<template>
  <MyButton
    background="#fff"
    color="#000"
    text="Hello World"
  />
</template>
<script>
import MyButton from "my-button";

export default {
  components:{
    MyButton
  }
}
</script>
```
